# 🔍 Lost and Found Portal

A simple and user-friendly web-based application to help students report, track, and recover lost items within a college campus.

---

## 📌 Overview

The **Lost and Found Portal** solves the common problem of misplaced belongings in large campuses. Instead of relying on scattered communication channels like WhatsApp, Instagram, or notice boards, this system provides a **centralized platform** for managing lost and found items efficiently.

---

## 🎯 Aim

To develop a web-based system that allows students to:
- Report lost or found items  
- Search and browse items بسهولة  
- Recover belongings quickly and efficiently  

---

## 🚀 Features

- 🔐 **User Authentication**
  - Login using registration number and password
  - Ensures only authorized users can access the system

- 📢 **Report Items**
  - Add details like item name, category, location, and description

- 🔎 **Browse & Search**
  - View all reported items
  - Real-time filtering using keywords

- 📂 **My Items**
  - Manage and view your own reported items

- ⚡ **Instant Filtering**
  - Search-based architecture for fast results

- 💾 **Local Storage Integration**
  - Data stored in browser using Local Storage API

---

  - Visual Studio Code

---

## 🧠 How It Works

1. User logs in using registration number  
2. Data is stored in **Local Storage**  
3. Users can:
   - Report items  
   - Browse items  
   - Search items instantly  
4. Data is dynamically displayed using JavaScript
